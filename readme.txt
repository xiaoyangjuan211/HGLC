# Dataset Instructions

1. Please download the following datasets from their official sources:
   - RAF-DB: http://www.whdeng.cn/RAF/model1.html
   - FERPLUS: https://www.kaggle.com/c/challenges-in-representation-learning-facial-expression-recognition-challenge/data
   - JAFFE: https://zenodo.org/record/3451592

2. Organize the data as follows:
   Dataset/
   ├── RAF-DB/
   │   ├── train/
   │   └── val/
   ├── FER2013/
